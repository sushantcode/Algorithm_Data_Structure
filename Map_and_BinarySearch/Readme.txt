#What it can do?
This program maintains counters and maps when changes are made. It also prints value in an order.

#Concepts Explored:
Binary Search
Sorting
Swapping

#Instructions
0-End Execution
1-print the counter table in ascending order of their indexes
2-print the counter table in ascending order of their values
3 x-add one to the counter of index x
4 x-subtract one from the counter of index x
5 x y-gives the number of counters whose values are larger than x and smaller y
