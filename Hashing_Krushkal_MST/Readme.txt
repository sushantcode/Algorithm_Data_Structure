#What it does?
This program prints the sum of the weights of the edges of the minimum spanning tree along with the trees involved and their respective leaders.

#Concepts used and explored:
Double Hashing
Graphs
Minimum Spanning Tree
Krushkal's Algorithm

#Instructions:
Enter the number of the vertices
Enter the number of the edges
Enter the tail-vertex name of a edge
Enter the head-vertex name of the edge
Repeat the last two steps
